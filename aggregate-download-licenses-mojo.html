license:aggregate-download-licenses

Full name:

org.codehaus.mojo:license-maven-plugin:1.17:aggregate-download-licenses

Description:

Download the license files of all aggregated dependencies of the current project, and generate a summary file containing a list of all dependencies and their licenses. Created on 23/05/16.

Attributes:

  • Requires a Maven project to be executed.
  • Executes as an aggregator plugin.
  • Requires dependency resolution of artifacts in scope: test.
  • Since version: 1.10.
  • Binds by default to the lifecycle phase: package.

  </licenseUrlReplacement>
